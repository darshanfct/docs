### How to Add New Documentation

1. **Create Project Folder**:
   Create a new folder in the `docs/` directory. Use a numbered prefix to control sorting.
   *Example:* `docs/03-internal-tools/`

2. **Add Markdown Files**:
   Add your content files inside the project folder. Use numbered prefixes for ordering.
   *Example:* `docs/03-internal-tools/01-overview.md`

3. **Publish**:
   Simply push your changes to GitHub.
   ```bash
   git add .
   git commit -m "docs: Add new internal tools docs"
   git push origin main
   ```

The system will automatically scan your new files, rebuild the index, and update the website within minutes.
