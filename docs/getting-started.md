## System Requirements

**Minimum Requirements:**
- Linux Server (Ubuntu 20.04 LTS or CentOS 8+)
- 8GB RAM
- 20GB Disk Space
- Docker & Docker Compose

**Recommended Setup:**
- Ubuntu 22.04 LTS or RHEL 8+
- 16GB RAM
- 100GB SSD Storage
- Kubernetes cluster (for high-availability)
